# Desafio Técnico - Frontend

Para o projeto funcionar será necessário primeiro iniciar o back 
Para rodá-lo, faça:

```console
> cd BACK
> npm install
> npm run server
```
E então rodar o front 
Para rodá-lo, faça:

```console
> cd FRONT
> npm install
> npm start
```
E Então abra no navegador através do endereço indicado no terminal.
